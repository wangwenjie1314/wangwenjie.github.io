目录结构

pc_ie6 pc端基于ui框架兼容ie6 构建快速模板

pc_bootstrap pc端基于bootstrap框架 构建快速模板

pc_gulp_ie6 pc端基于gulp(旧版本，仅支持windows[将项目中node_modules压缩包解压至该目录下])+sass+ui兼容ie6 构建快速模板

pc_gulp_ie6_adv pc端基于gulp(新版本，windows/liunx/mac均可)+sass+ui兼容ie6 构建快速模板

pc_gulp_require_ie6_adv pc端基于gulp(新版本，windows/liunx/mac均可)+require+sass+ui兼容ie6 构建快速模板

pc_gulp_bootstrap pc端基于gulp(旧版本，仅支持windows[将项目中node_modules压缩包解压至该目录下])+sass+bootstrap 构建快速模板

pc_gulp_bootstrap_adv pc端基于gulp(新版本，windows/liunx/mac均可)+sass+bootstrap 构建快速模板

pc_gulp_require_bootstrap_adv pc端基于gulp(新版本，windows/liunx/mac均可)+require+sass+bootstrap 构建快速模板

wap_amaze wap端基于amaze框架 构建快速模板

wap_normal wap端无框架 构建快速模板

wap_gulp_amaze wap端基于gulp(旧版本，仅支持windows[将项目中node_modules压缩包解压至该目录下])+sass+amaze框架 构建快速模板

wap_gulp_amaze_adv wap端基于gulp(新版本，windows/liunx/mac均可)+sass+amaze 构建快速模板

wap_gulp_noraml wap端基于gulp(旧版本，仅支持windows[将项目中node_modules压缩包解压至该目录下])+sass 构建快速模板

wap_gulp_normal_adv wap端基于gulp(新版本，windows/liunx/mac均可)+sass 构建快速模板

wap_gulp_require_normal_adv wap端基于gulp(新版本，windows/liunx/mac均可)+require+sass 构建快速模板

特殊说明

以上目录中含 gulp字符的文件夹，需要进入该目录
然后在控制台执行以下命令才可以正常运行(需要node npm)：

//先拉取运行所需要的环境

npm install --save--dev

//拉取环境成功以后 执行开发环境跑起来//自动打开浏览器 自动刷新改动

gulp serve

//开发结束后 产出静态页面文件给后台技术 或者自己使用

gulp default

》》如果 gulp serve 后报错，那么 请记得修改 gulpfile.js 里面的 myComputerIp(不清楚那么请往下看) ;

另 手机端wap开发基于gulp开发，

如果需要手机端开发看下页面效果，那么需要手机和电脑在同一个路由器网络下，然后查看电脑ip地址（以作者的10.58.139.169为例），那么手机端浏览器的地址为：http://10.58.139.169:9000/ （9000为gulp配置默认端口）

如果需要开启手机端调试模式，那么需要配置下[wap_gulp_amaze] 和 [wap_gulp_noraml] 目录下"gulpfile.js"中的 weinre任务

var myComputerIp="10.58.139.169";//修改为你开发环境所在的电脑ip
